# рҹҡҖ Installation

## Backend

```bash
cd backend
pip install -r ../requirements.txt
python manage.py migrate
```

---

## Run ETL

```bash
python etl/01_extract_from_mysql.py
python etl/02_clean_and_transform.py
python backend/manage.py load_financials
```

---

## Frontend

```bash
cd frontend
npm install
npm run dev
```

Launch:

```text
http://localhost:5173
```
